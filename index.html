üíª Installing Visual Studio Code (VS Code)

VS Code is the recommended code editor for this course because it provides excellent Python support, integrated terminal, and debugging tools.

Installation Steps:

Windows:

  1. Go to https://code.visualstudio.com/
  2. Click "Download for Windows"
  3. Run the installer (.exe file)
  4. Important: During installation, check both:
    • ‚úÖ "Add to PATH" (recommended)

  1. Go to https://code.visualstudio.com/
  2. Download for macOS
  3. Open the downloaded .zip file

Linux (Ubuntu/Debian):

üì• Installing Python 3.12

To download Python 3.12, visit the official Python website and follow the installation instructions for your operating system.

Steps to Download and Install Python 3.12

  1. Go to the official website: Open your web browser and navigate to the Python Downloads page.
  2. Select your OS: The website often automatically detects your operating system (Windows, macOS, or Linux) and offers the appropriate download. If not, you can manually select your OS tab.
  3. Download the installer: Click the button with the latest Python 3.12 version number (e.g., Python 3.12.x) to download the installer file.
    • For Windows/macOS: This will typically be an executable (.exe) file for Windows or a disk image (.pkg or universal2) for macOS.
    • For Linux: You can download the source code, or more commonly, use your operating system's package manager with specific commands (e.g., on Ubuntu, you might need to add a repository and install via apt).
  4. Run the installer:
    • Windows: Open the downloaded .exe file. Crucially, in the first screen of the installation wizard, ensure you check the box that says "Add python.exe to PATH" to ensure the python command is recognized in your command prompt. You may also want to select "Install for all users" and choose "Customize installation" to review optional features like pip and IDLE, which are recommended.
    • macOS: Open the downloaded package file and follow the prompts in the installer.
  5. Verify the installation: After the installation is complete, open your terminal or Command Prompt and type the following command to confirm the installed version:

python --version
# or sometimes
python3 --version

This should display the installed version, such as Python 3.12.x.

# Navigate to your project folder
cd C:\Users\teamp\Downloads\testtt

python -m venv .venv

# Activate it

cd ~/Projects/testtt

# Create virtual environment
python3 -m venv .venv

# Activate it
source .venv/bin/activate

You should see (.venv) at the beginning of your terminal prompt when activated.

Step 2: Install OpenCV

OpenCV (Open Source Computer Vision Library) is used for:

  • Reading images and video from camera
  • Image processing (resizing, color conversion)
  • Drawing shapes and text on images
  • Displaying images in windows

Install Command:


pip install opencv-python

Expected output:


Collecting opencv-python
  Downloading opencv_python-4.x.x.x-cp312-cp312-win_amd64.whl
Installing collected packages: numpy, opencv-python
Successfully installed opencv-python-4.x.x.x numpy-1.x.x

Verify installation:


python -c "import cv2; print(cv2.__version__)"

Should print something like: 4.8.1 (version number)

Step 3: Install MediaPipe

MediaPipe is Google's machine learning framework that provides:

  • Pre-trained models for hand detection
  • Pose detection and face detection
  • Real-time processing capabilities
  • Easy-to-use Python API

Install Command:


pip install mediapipe

Expected output:


Collecting mediapipe
  Downloading mediapipe-0.x.x-cp312-cp312-win_amd64.whl
Installing collected packages: mediapipe
Successfully installed mediapipe-0.x.x

Verify installation:


python -c "import mediapipe as mp; print(mp.__version__)"

Should print something like: 0.10.9 (version number)

Step 4: Install All Dependencies at Once (Alternative Method)

If you prefer, you can create a requirements.txt file with all dependencies:

Create requirements.txt:

txt
opencv-python>=4.8.0
mediapipe>=0.10.0

Install from requirements.txt:


pip install -r requirements.txt

This will install both packages at once.

Step 5: Verify Everything is Installed

Common Installation Issues

Issue: "pip is not recognized"

Solution:

  1. Make sure virtual environment is activated
  2. OR use: python -m pip install opencv-python mediapipe

Issue: "Permission denied" (Windows)

Solution:


# Run PowerShell as Administrator, or
# Add --user flag
pip install --user opencv-python mediapipe

Issue: "No module named 'cv2'" after installation

Solution:

  1. Make sure you're using the virtual environment Python
  2. In VS Code: Press Ctrl+Shift+P ‚Üí "Python: Select Interpreter" ‚Üí Choose .venv
  3. Restart terminal and activate .venv again

Issue: Installation takes too long

Solution:

  1. Be patient - opencv-python is ~90MB
  2. Check your internet connection
  3. Try using a faster pip mirror (optional):

pip install --upgrade pip
pip install opencv-python mediapipe --no-cache-dir

Issue: Wrong Python version

Solution:


# Check Python version (should be 3.12+)
python --version

# If wrong version, specify Python 3.12 explicitly
py -3.12 -m venv .venv
