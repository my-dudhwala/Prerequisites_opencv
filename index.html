Prerequisites

Before installing the libraries, make sure you have:

  • ‚úÖ Python 3.12 installed
  • ‚úÖ Virtual environment created and activated
  • ‚úÖ VS Code open with your project folder

Step 1: Create Virtual Environment (If Not Already Created)

Windows PowerShell:

powershell
# Navigate to your project folder
cd C:\Users\teamp\Downloads\testtt

# Create virtual environment
python -m venv .venv

# Activate it
.\.venv\Scripts\activate

macOS/Linux:

bash
# Navigate to your project folder
cd ~/Projects/testtt

# Create virtual environment
python3 -m venv .venv

# Activate it
source .venv/bin/activate

You should see (.venv) at the beginning of your terminal prompt when activated.

Step 2: Install OpenCV

OpenCV (Open Source Computer Vision Library) is used for:

  • Reading images and video from camera
  • Image processing (resizing, color conversion)
  • Drawing shapes and text on images
  • Displaying images in windows

Install Command:

bash
pip install opencv-python

Expected output:


Collecting opencv-python
  Downloading opencv_python-4.x.x.x-cp312-cp312-win_amd64.whl
Installing collected packages: numpy, opencv-python
Successfully installed opencv-python-4.x.x.x numpy-1.x.x

Verify installation:

bash
python -c "import cv2; print(cv2.__version__)"

Should print something like: 4.8.1 (version number)

Step 3: Install MediaPipe

MediaPipe is Google's machine learning framework that provides:

  • Pre-trained models for hand detection
  • Pose detection and face detection
  • Real-time processing capabilities
  • Easy-to-use Python API

Install Command:

bash
pip install mediapipe

Expected output:


Collecting mediapipe
  Downloading mediapipe-0.x.x-cp312-cp312-win_amd64.whl
Installing collected packages: mediapipe
Successfully installed mediapipe-0.x.x

Verify installation:

bash
python -c "import mediapipe as mp; print(mp.__version__)"

Should print something like: 0.10.9 (version number)

Step 4: Install All Dependencies at Once (Alternative Method)

If you prefer, you can create a requirements.txt file with all dependencies:

Create requirements.txt:

txt
opencv-python>=4.8.0
mediapipe>=0.10.0

Install from requirements.txt:

bash
pip install -r requirements.txt

This will install both packages at once.

Step 5: Verify Everything is Installed

Run this quick test in your terminal:

bash
python -c "import cv2; import mediapipe as mp; print('‚úÖ OpenCV version:', cv2.__version__); print('‚úÖ MediaPipe version:', mp.__version__); print('All packages installed successfully!')"

Expected output:


‚úÖ OpenCV version: 4.8.1
‚úÖ MediaPipe version: 0.10.9

Common Installation Issues

Issue: "pip is not recognized"

Solution:

  1. Make sure virtual environment is activated
  2. OR use: python -m pip install opencv-python mediapipe

Issue: "Permission denied" (Windows)

Solution:

powershell
# Run PowerShell as Administrator, or
# Add --user flag
pip install --user opencv-python mediapipe

Issue: "No module named 'cv2'" after installation

Solution:

  1. Make sure you're using the virtual environment Python
  2. In VS Code: Press Ctrl+Shift+P ‚Üí "Python: Select Interpreter" ‚Üí Choose .venv
  3. Restart terminal and activate .venv again

Issue: Installation takes too long

Solution:

  1. Be patient - opencv-python is ~90MB
  2. Check your internet connection
  3. Try using a faster pip mirror (optional):
bash
pip install --upgrade pip
pip install opencv-python mediapipe --no-cache-dir

Issue: Wrong Python version

Solution:

bash
# Check Python version (should be 3.12+)
python --version

# If wrong version, specify Python 3.12 explicitly
py -3.12 -m venv .venv
